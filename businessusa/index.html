Visitor Login Flow

  1. BusinessUSA
    • http://business.usa.gov/access-financing
  2. BusinessUSA > Login
    • http://my.business.usa.gov/signin/?return_to=business.usa.gov/access-financing
  3. MyUSA Login
    • http://my.usa.gov/sign_in
  4. BusinessUSA
    • http://business.usa.gov/access-financing?uid=1234&latitude=32.4030000&longitude=-102.494778&token=abcdef


Subsequent API request for Business Profile:
http://my.business.usa.gov/api/profiles?uid=1234&token=abcdef
